# Minsta
An Instagram demo whose source come from [500px.com](https://500px.com)

### Features

* iOS 8 or later;
* AsyncDisplayKit 1.9.4;
* Only read opeation is permitted.

### Usage

* Install pods, and build project in Xcode.
